# Synacor VM

Synacor challenge https://challenge.synacor.com/

## The source code

This is my first project in golang!

## Build it

```shell
$ go build ./src/vm.go
$ ./vm challenge.bin
```

or you can run it without building it

```shell
$ go run ./src/vm.go challenge.bin
```

## Test it

```shell
$ go test ./src/synacor
```

That's all!
